# JH-WallCraft

🎨 一个纯前端的日程壁纸生成器，支持自定义弥散光背景、多屏幕导出、自动日期更新。

![preview](preview.png)

## ✨ 功能特性

- **背景定制**
  - 纯弥散光渐变
  - 纯自定义图片
  - 弥散光 + 图片叠加
  - 4种预设色系 + 自定义颜色
  - 光源数量（1-3个）
  - 光源位置（9宫格选择）
  - 光源强度调节

- **日历系统**
  - 自定义日期范围
  - 倒计时目标
  - 时间线事件（空心边框）
  - 关键节点（实心标签）
  - 特殊标记（带图标和时间）

- **待办管理**
  - 今日待办列表
  - 勾选/删除

- **多屏幕支持**
  - 自定义多个屏幕分辨率
  - 批量导出所有屏幕壁纸

- **自动更新**
  - 每分钟检测日期变化
  - 跨天自动重新生成
  - 浏览器通知提醒

- **数据持久化**
  - 所有配置保存到 localStorage
  - 刷新页面自动恢复

## 🚀 使用方法

### 方式一：直接打开

双击 `index.html` 即可在浏览器中使用。

### 方式二：本地服务器

```bash
# Python
python -m http.server 8080

# Node.js
npx serve
```

然后访问 `http://localhost:8080`

### 方式三：设为浏览器启动页

将 `index.html` 的完整路径设为浏览器启动页，每次开机自动打开并更新壁纸。

## 🛠️ 技术栈

- 纯 HTML/CSS/JavaScript
- Canvas 2D 绘图
- localStorage 数据存储
- 无任何外部依赖

## 📝 使用建议

1. **日常使用**：保持浏览器标签页常驻，跨天自动更新
2. **多屏幕**：在「屏幕配置」添加各显示器分辨率，点击「导出全部」批量下载
3. **自定义背景**：上传喜欢的图片，选择「弥散光+图片叠加」获得最佳效果
